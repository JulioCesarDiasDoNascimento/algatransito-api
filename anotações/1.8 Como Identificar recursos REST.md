# Anotações 
## URI(Uniform Resource Identifier)

* Conjuto de caracteres que identifica um recurso na web
* URL(Localização do recurso(ENDEREÇO COMPLETO)) é um tipo de URI(Somente o recurso)

> Exemplo: Coleções
> * /produtos é a URI
> * https://algaworks.com.br/produtos/ 

> Exemplo: Unico
> * /produto/{codigo} é a URI
> * https://algaworks.com.br/produto/331 

Ideal sempre usar os nomes do recursos no plural!
