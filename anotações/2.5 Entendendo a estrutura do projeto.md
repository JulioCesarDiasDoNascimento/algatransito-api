# Anotações 
## Pontos importantes sobre a estrutura do projeto

* A pasta .mvn é um wrapper do maven(serve para não precisar instalar o maven na máquina)

* A pasta src é onde fica o codigo do projeto
* src/main/java
* src/main/resources(configurações)

* O gitignore serve para ignorar arquivos que não queremos subir para o repositório remoto

> POM(Project Object Model) 
> * É o coração do maven
> * Contem informações sobre o projeto, para build e dependencias
> * O parant é uma herança
> * O pom é um xml(Baseado em tag), pode existir tag dentro de tag

## Maven 
* É uma ferramenta de automação de build
* mvn clean -> limpa os arquivos compilados
* mvn validate -> valida o projeto
* mvn compile -> compila o projeto
* mvn test -> executa os testes unitários
* mvn package -> empacota o projeto em um formato distribuível(jar, war)
* mvn install -> instala o pacote no repositório local

> fatJAr -> JAR com todas as dependencias, é o projeto compilado(depois do mvn package)
> * como rodar o arquivo jar?:
> * java -jar algatransito-api-0.0.1-SNAPSHOT.jar
